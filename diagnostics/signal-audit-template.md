Signals represent the attributes AI agents use to evaluate and compare brands.  
This template helps teams evaluate whether those signals are accurate, consistent, and trustworthy.

---

## Signal Types

- Pricing  
- Availability  
- Location  
- Reviews  
- Performance metrics  
- Guarantees  
- Safety statements  
- Product/service specifications  
- Compliance attributes  

---

## Audit Checklist

### Accuracy
- [ ] Data matches live product/service  
- [ ] Versioning is up to date  
- [ ] No stale information  

### Completeness
- [ ] All required attributes available  
- [ ] Comparison-critical attributes included  

### Consistency
- [ ] Same value across all systems  
- [ ] No conflicting representations  

### Reliability
- [ ] Data sourced from single source of truth  
- [ ] Automated refresh processes in place

- Signal Quality Score
