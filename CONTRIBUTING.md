# Contributing to the Invocation Framework

The Invocation Framework is maintained by Imvara as part of its mission to help global organizations become invokable by AI systems.

We welcome contributions that:

- Improve machine-readability  
- Strengthen definitions and terminology  
- Expand diagnostic templates and patterns  
- Add additional examples demonstrating Invocation  
- Improve technical specifications  

## Contribution Requirements

All contributions must:

1. Align with the Invocation Architecture principles  
2. Maintain clarity, structure, and accuracy  
3. Be vendor-neutral and industry-agnostic  
4. Follow the Imvara style guide (clarity > jargon)

## How to Contribute

1. Fork the repository  
2. Create a new branch  
3. Submit a pull request with a clear explanation  
4. The Imvara maintainers will review submissions
